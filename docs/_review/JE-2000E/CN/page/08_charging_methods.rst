太阳能充电
==========

本产品配备两个 DC8020 直流输入端口，最大支持 800W 输入，可搭配电小二 SolarSaga 系列太阳能板使用。

直接连接
--------

.. image:: _assets/templates/word_template/common_assets/charging/solar_direct.png
   :alt: Direct solar charging placeholder image.
   :width: 360px

如果单个 DC8020 直流输入端口需要同时连接两个太阳能板，请使用太阳能串联转接盒（单独售卖）。

转接盒连接
----------

.. image:: _assets/templates/word_template/common_assets/charging/solar_adapter.png
   :alt: Solar charging adapter placeholder image.
   :width: 360px

注意
~~~~

- 单个 DC8020 输入端口请勿连接两个以上光伏板。
- 同时连接到两个 DC 输入端口的电源，其电压必须保持一致，否则可能导致产品异常或损坏。
- 连接太阳能板至两个 DC8020 输入端口时，请使用相同型号的太阳能板，并确保连接数量一致。
- 给产品充电时，请勿同时使用车载充电与光伏充电，否则可能烧坏车内保险丝或导致无法充电。

建议使用电小二官方太阳能板为本产品充电。请确保所用太阳能板的开路电压（V~oc~）位于 16V 至 60V 的直流输入电压范围内。因使用第三方太阳能板或配件导致的损坏或损失，电小二概不负责。

车充充电
--------

本产品适用于 12V 车型车充口进行充电。请确保汽车车充口和车充线的点烟器接触良好，并确保点烟器完全插入到位。

.. image:: _assets/templates/word_template/common_assets/charging/car_charge.png
   :alt: Car charging placeholder image.
   :width: 360px

.. note::

   车载充电线需单独购买。

注意
~~~~

- 请在汽车点火启动后使用车充充电，以免造成汽车电瓶亏电无法启动。
- 汽车在行驶过程中如路况颠簸，禁止使用车充充电，以防止接触不良引起接触点烧坏。若因不按规范操作造成损失，本公司不承担相应责任。
- 车充充电仅支持 12V 车型，不支持 24V 车型。请勿使用 24V 车型为本产品充电，以免造成人身伤害及财产损失。
