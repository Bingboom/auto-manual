保修说明
========

感谢您购买电小二产品。

激活与服务
----------

您可以通过微信注册产品，激活质保服务。注册后可享有以下服务：

- 自购买日起享质保 3 年，注册延保 2 年
- 电小二售后专家技术支持
- 真伪查询服务
- 售后维修进度实时查看

产品具体保修条款和注册细则，请登录 **电小二户外电源服务小程序** 查看。

不在保修范围内的情况
--------------------

以下情况不在保修范围内：

- 产品使用后出现的外观损坏
- 自行拆机或维修
- 因人为原因造成的产品性能故障
- 由于自然灾害、雷电、事故等不可抗拒因素造成的损害
